# SikolethuStudios
Photography Portfolio Website
Project Overview

This project is a modern, responsive photography portfolio website developed for a client to showcase their photography work, services, and contact information. The website is designed to provide a clean visual experience, highlight featured projects, and support professional online presence.

The site emphasizes performance, accessibility, and scalability, ensuring it can grow with the clientâ€™s brand and content needs.

Features

Responsive design for desktop, tablet, and mobile devices

Modern, clean UI optimized for visual content

Portfolio section to showcase photography work

About and Services sections

Blog section (optional / extensible)

Contact section for client inquiries

Reusable and well-structured React components
Tech Stack

Frontend Framework: React (TypeScript)

Build Tool: Vite

Styling: CSS (global styles and component-based styling)

Package Manager: npm

Version Control: Git & GitHub

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

**Installation & Setup (For Developers)**


Node.js (v18 or later recommended)

npm

Steps

Clone the repository:

git clone https://github.com/<your-username>/<repository-name>.git


Navigate into the project directory:

cd photography-portfolio-website


Install dependencies:

npm install


Start the development server:

npm run dev


Open the local URL provided in the terminal to view the project.

Build for Production

To generate a production-ready build:

npm run build


The output will be created in the dist/ directory.
